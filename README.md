## 🎓 Online Examination System

A full-stack web application for conducting secure, timed, and randomized MCQ exams with instant results.

---

## 🔍 Overview

The system allows students to take online exams with:

- Timed sessions with automatic submission
- Randomized questions and answer choices per attempt
- Instant grading upon submission
- One-attempt enforcement (no retakes)

---

## ⚙️ Key Features

- ⏱️ **Real-Time Countdown Timer**  
  Exams are automatically submitted when time expires.

- 🔀 **Randomized Questions & Answers**  
  Each attempt generates a unique order to prevent cheating.

- 📊 **Automated Grading**  
  Scores are calculated instantly using SQL Server Stored Procedures.

- 🔒 **One Attempt Policy**  
  Enforced at both application and database levels.

- 🏗️ **Clean Architecture**  
  Separation of concerns using Service Layer, Interfaces, and ViewModels with thin controllers.
